# Cryptopals
Solutions to the cryptography challenges found @ https://cryptopals.com

All solutions written in Python 3.

The Cryptopals Crypto challenges are 64 exercises divided into 8 sets. As the Cryptopals website states, " They're derived from weaknesses in real-world systems and modern cryptographic constructions. We give you enough info to learn about the underlying crypto concepts yourself. When you're finished, you'll not only have learned a good deal about how cryptosystems are built, but you'll also understand how they're attacked."
